## 开发
### 依赖环境
- [Node.js v16.20.1 < v18*](https://nodejs.org/)

### 安装依赖
```shell
npm i
```

### 本地预览
```shell
npm run docs:dev
```

### 构建静态页面
构建完成的网页会保存在 ``src\.vuepress\dist``
```shell
npm run docs:build
```
